Craft 5.x In 5.9.0, we improved the accessibility of user permission lists and GraphQL schema component lists . Previously, “Select All” buttons to select all checkboxes in the list were only accessible via pointer devices. They have been updated to use semantic buttons and provide additional descriptive information via aria-describedby .
